## Run Locally

Clone the project

```bash
  git clone https://github.com/MaxBenDuah/bookmarkly-project-tailwind.git
```

Go to the project directory

```bash
  cd bookmarkly-project-tailwind
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run watch
```
